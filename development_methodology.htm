Our Development Methodology
 

Our Development Methodology is based on the following principles :-

  • Establish a sound Foundation.
  • Follow structured Steps using Controlled Evolution.
  • Aim for high User involvement.

    • THE MAJOR STEPS
    STEP DESCRIPTION COMMENT
    1 Create the Data Models The Methodology is Database-driven.
    2 Check the Library of Page Templates The Library should be comprehensive.
    3 Populate any appropriate Tables. Using either ASP or Access.
    4 Agree the User Scenarios and the detailed Steps. User involvement is vital.
    5 Define the Deliverables and the Time-box. User involvement is vital.
    6 Create any required HTML. To achieve tailored Pages.
    7 Agree the deliverables with the Users. There should be no nasty surprises.
