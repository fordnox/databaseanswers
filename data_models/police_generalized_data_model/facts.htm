The Area being Modelled is :-
A Generalised Database for any Police Service

Source material for this work was originally taken from the UK Pito Corporate Data Model

A. THE USER REQUIREMENTS :
A Generalised Database that will apply to any Police Force, anywhere in the world.

B. THE APPROACH :
The Approach was to start with the Generic Data Model for Organisations and People, 
and map the Police Force Database on to this Data Model.
