Wikipedia has an entry for Drug Class in which it says :-
"A drug class is a set of medications that have :-
  1. similar chemical structures
  2. the same mechanism of action
  3. a related mode of action
  4. and/or are used to treat the same disease.
In several dominant drug classification systems, these four types of classifications form a hierarchy".
