Possible User Scenarios for Communities
 

Sample Scenarios include the following :-

  • A New User registers.
  • A Registered User sets up a new Community as Moderator.
  • A Registered User joins a Community of Interest.
  • A Community Member posts a Message on the Bulletin Board.

    • DETAILS OF USER SCENARIOS
    Nr. WHO ACTIVITY DESCRIPTION TABLES USED
    1 New User Register complete the Online Registration Users
    2 Registered User Create a Community Complete the Online Procedure. Users, Communities
    3 Registered User Join a Community Complete the Online Procedure. Users(R), Communities(R), Users_in_Communities,(C,U)
    3 Community Member Use the Bulletin Board Put a Message on the Bulletin Board. Users, Messages.
