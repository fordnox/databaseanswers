Data Model for Citizens   
An Access Database is available on demand.

We might design a Data Warehouse, which would be very useful for providng data for Reports and Business Intelligence but is not relevant.

We would be glad to have your comments.

If you are new to Data Models, this page of my new Tutorial will help you understand the Data Model.

This Model is aligned with the Entity-Attribute-Value Approach, which is a very flexible, open-ended design.
It is ideal for this kind of situation, where every Citizen can have an entirely different set of data, depending on the Events in their lives.
