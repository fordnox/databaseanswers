FACTS for the Data Model for Athletes
 

A diagram showing a Conceptual Data Model is available. 

The Scope of the Data Model is Athletes competing in Events.
A. The Things of Interest :-
A.1 Athletes 
A.2 Athlete Categories
A.3 Clubs
A.4 Gender 
A.5 Events
A.6 Series of Events

B. How are the Things of Interest related ?
B.1 Each Athlete has a Category.
B.2 Each Athlete has a sex (Gender).
B.3 Each Athlete particpates in an Event.
B.4 An Event can be split up into a Series of Events 
    if there are too many Athletes for one Event. 
B.5 Each Athlete is associated with one and only one Club.

C. What do we know about the Things Of Interest ?
C.1 An Athlete has a name, and date of birth. 
C.2 An Athlete Category can be either Senior or Junior.
C.3 An Event Category is the same as an Athlete Category.
C.4 A Gender can be either Male or Female.
