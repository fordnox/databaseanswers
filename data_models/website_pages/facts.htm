Data Model for Web Site Pages
 

The FACTS define the Business Rules ...

Each WebSite Page can be derived from a Template, but this is not necessary.

Each Page is made up of a number of elements :-

  1. Images
    • Each Image is one of a defined Type,(e.g. Button, Banner, Question, etc.)
  2. Paragraphs
    • Each Paragraph is one of a defined Type,(e.g. Answer, Header, News, Top Tip, etc.)
  3. Lines
    • Each Line is one of a defined Type,(e.g. Bullet Points, Intoductory copy, Page Heading, Question.)
