Data Model for User Tables
 
Back to the Conceptual Data Model, and an Access Database is available on demand.


Here's the Script to create the Tables in Oracle ... 
 
CREATE TABLE DD_Users(
 user_id         INTEGER PRIMARY KEY,
 user_login_name VARCHAR2(30),
 user_category   CHAR(15),
 user_registration_date DATE,
 user_email      VARCHAR2(80),
 user_country    VARCHAR2(80),
 user_language   VARCHAR2(80))
/

CREATE VIEW VW_Users AS
SELECT
user_login_name ,
user_category   ,
user_registration_date,
user_country    ,
user_language   ,
user_email 
FROM  table_name -- eg dd_users
/
