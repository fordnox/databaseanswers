Background to Implementation of a CRM Pilot Project 
IMPLEMENTATION OF A PILOT PROJECT

Purpose
The purpose of the Pilot Project is to establish the Proof-of-Concept for the 
design and implementation of the Data Architecture.

In other words, to verify that the Data Architecture, implemented using the 
components in the appropriate Software Toolkit, will support the functionality in 
the planned Project.

The Deliverable will be the result of a Design Review of the Pilot Project.
The Pilot Project will be based on a number of User Scenarios.
