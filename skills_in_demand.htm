Update on 10th. April 2003 :-
Review the Jobs Check in the US and the UK, taken on March 26th. 2002.

I have been working as a Data Architect for the past 2 years but now that I have finished,
I find that not many organizations recognize the need and value of that role.
So it's hard to recommend it as a career direction.
What I do notice is a demand for Data Analysts, very often in the context of Data Migration from
legacy systems to ERPs like SAP/Siebel (often combined), or Data Integration of existing systems.

The only major changes in key skills is an increasing demand for SQL Server expertise, Java has faded,
Microsoft has more clearly defined their strategy of going their own way with .net and C#, and of course,
e-commerce skills have died away to be replaced by Web Services.
Summary of What's in Demand - 30th. March, 2001
Oracle is, (as always), the dominant Database skill required.
This week, CRM is in great demand, with Siebel and BroadVision as specifics.
Siebel is twice as popular as BroadVision right now, but we'll track them to see if BroadVision is gaining.
