Website Development Plan 
This page describes plans for Website Development.
  • The planned Phases for User Login are shown in this Table.
  • Here is a useful Data Model for User Portals

    WELCOME
     
    1. User Login as at present Everybody is classified as Public, and the same Home Page is presented to everybody with the ability to Add or Remove Links.
    2. Plus User Tracking. This facility will track the pages that each User visits and then add them to his available Links in order of popularity.
    3. Plus User Roles. The User can choose from a drop-down list of Roles, such as DBA or Data Analyst. He will be able to add a new Role and select it.
    4. Plus a Generic Report facility. This will let the User enter an Excel Spreadsheet and request a variety of Reports to be produced.
    It will be based on a Generic Data Mart and Report (perhaps from Microsoft SQL Server Reporting Services). with mapping from the specific Spreadsheet.
    5. Plus Site Content and Tags. The Site Content and Tags will tell the Visitor about the Content and Structure of the information in the WebSite.
    6. Plus User Interface and Vocabulary. The User Interface and Vocabulary will let the Visitor choose from a Menu of options, tailored to his Role.
    7. Plus Gold, Silver or Bronze Membership Levels. The User can choose from a drop-down list of Roles, such as DBA or Data Analyst.
    This will require a payment mechanism so that Users will pay a monthly fee for their membership.
    This could probably be done using Paypal's Subscription facility.
