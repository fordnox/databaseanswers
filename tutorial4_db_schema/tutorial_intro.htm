Tutorial on "Understanding a Database Schema" - Introduction

Back   First Slide

  1. What is the purpose of this Tutorial ?
    The purpose is to help you :-
    • To understand a Database Schema.
    • To extend an existing Schema.

  2. Background
    • This Tutorial follows common conventions in Data Modelling.
    • It explains Primary and Foreign Keys briefly.
    • It explains Inheritance and Many-to-Many Relationships.
    • It uses Diagrams produced by a Software Product specially designed for Data Modelling.

  3. The Approach
    • The Approach I have adopted is to use a situation that many people are familiar with.
    • This is a basic Customer Ordering System.
    • Then I decided to start simple and progress to something more complex.
    • This means starting with Starbucks and then moving Amazon.com.

  4. The Scope
    • In the Scope I have included the basic topics you need to know about Database Schema :-
      • Primary and Foreign Keys.
      • One-to-Many and Many-to-Many Relationships
      • Product Inheritance.
      • Recursive relationships and Rabbit's Ears.
